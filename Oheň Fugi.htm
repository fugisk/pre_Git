Oheň

08.08.2021 17:59

Horčíkové kresadlo - zaujímavá vec. Len sa mi s ňou nepodarilo založiť oheň v prípade, že som mal k dispozícii iba suché drevo.

    Čo fungovalo - vykresanie iskry - kresadlo + kovový vrchnák od nápojov ( pivo, minerálka ...; alkohol nie  ;-)  )

                                                     kresadlo + čepele nožov; klince ( niečo s ostrou hranou )

                                                     kresadlo + sklenený črep

                        - vzbĺknutie - rôzne chumáče, tzv. hniezda ( cupanina špagátu, vata, mach, seno, brezová kôra, vyklepaný choroš )
